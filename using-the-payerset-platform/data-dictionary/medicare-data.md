# Medicare Data

Medicare data in Payerset is split into separate dictionaries for inpatient and outpatient analysis.

* Medicare Inpatient Data
* Medicare Outpatient Data
* Medicare Inpatient Methodology

Use the inpatient dictionary for IPPS-based hospital reimbursement fields.

Use the outpatient dictionary for HCPCS, RVU, locality, and OPPS-related fields.
